## What is covered in Sprint 3

Sprint 3 covers the full transactional layer of TutoConnect — communication, booking, payments, and reviews.  
The prototype demonstrates the end-to-end experience for students booking sessions, messaging tutors, making payments, and leaving reviews, as well as admin tools for monitoring and moderation.

## Screens included

- Real-Time Messaging (Student & Tutor chat)
- Session Booking — 1-on-1
- Session Booking — Group
- Calendar View (available slots)
- Video Call Link Generation
- Payment Page
- Payment Confirmation & Invoice Download
- Post-Session Review Form
- Star Rating Submission
- Admin Payment Overview
- Admin Flagged Content Moderation
- Admin Analytics Dashboard
- Admin Announcements Panel

## How to use the prototype

1. Open the link above
2. Click the **Play** button to enter present mode
3. Click through the screens using the interactive buttons and links

## Main flows to test

| Flow | Start | End |
|---|---|---|
| Send a message | Messaging Screen | Message Delivered |
| Book a 1-on-1 session | Calendar View | Booking Confirmation |
| Book a group session | Group Session Screen | Booking Confirmation |
| Complete a payment | Payment Page | Invoice Download |
| Leave a review | Post-Session Prompt | Review Submitted |
| Admin monitor payments | Admin Payment Overview | Payment Detail |
| Admin moderate content | Flagged Content List | Content Removed |
| Admin publish announcement | Announcements Panel | Announcement Live |
