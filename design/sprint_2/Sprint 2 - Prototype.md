## What is covered in Sprint 2

Sprint 2 covers the dashboards, profiles, settings, and admin management features of TutoConnect.  
The prototype demonstrates the main interfaces used by students, tutors, and administrators after signing in.

## Screens included

- Student Dashboard
- Tutor Dashboard
- Admin Dashboard
- Tutor Public Profile — About Tab
- Tutor Public Profile — Reviews Tab
- Student Own Profile
- Student Profile — Edit State
- Settings Panel
- Deactivation Confirmation Modal
- Admin User Management
- Admin Deactivation Confirmation Modal

## How to use the prototype

1. Open the link above  
2. Click the **Play** button to enter present mode  
3. Click through the screens using the interactive buttons and links
