## What is covered in Sprint 1

Sprint 1 covers the authentication and search features of TutoConnect. The prototype includes 10 clickable screens demonstrating the core user flows.

## Screens included

- Registration Page (common signup)
- Student Registration
- Tutor Registration
- Sign In
- Search Results
- Filter Popup
- Recover Password
- OTP Verification
- Update Password
- Side Menu

## How to use the prototype

1. Open the link above
2. Click the **Play** button to enter present mode
3. Click through the screens using the interactive buttons and links

## Main flows to test

| Flow | Start | End |
|---|---|---|
| Student Registration | Registration Page | Sign In |
| Tutor Registration | Registration Page | Sign In |
| Login | Sign In | Search Results |
| Forgot Password | Sign In → Recover Password → OTP → Update Password | Sign In |
| Filter | Search Results → Filter Popup → Apply | Search Results |
| Logout | Side Menu → Log out | Sign In |
