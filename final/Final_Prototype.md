## What is covered in the Final Prototype

The Final Prototype is the complete, consolidated design of TutoConnect covering all three sprints.  
It represents the full platform experience across all three user roles — Student, Tutor, and Administrator — from registration through to communication, booking, payment, and review.

## How to use the prototype

1. Open the link above
2. Click the **Play** button to enter present mode
3. Click through the screens using the interactive buttons and links

## Main flows to test

| Flow | Start | End |
|---|---|---|
| Student registration | Sign In | Student Dashboard |
| Tutor registration | Sign In | Tutor Dashboard |
| Forgot password | Sign In | OTP Verified — Password Reset |
| Search & book a tutor | Student Dashboard | Booking Confirmation |
| Complete a payment | Payment Page | Invoice Download |
| Send a message | Messaging Screen | Message Delivered |
| Leave a review | Post-Session Prompt | Review Submitted |
| Edit student profile | Student Own Profile | Profile Updated |
| Manage settings | Settings Panel | Changes Saved |
| Admin approve tutor | Admin User Management | Tutor Approved |
| Admin deactivate user | Admin User Management | Account Deactivated |
| Admin monitor payments | Admin Payment Overview | Payment Detail |
| Admin moderate content | Flagged Content List | Content Removed |
| Admin publish announcement | Announcements Panel | Announcement Live |
